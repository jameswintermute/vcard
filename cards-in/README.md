# cards-in/

Drop your exported `.vcf` contact files here before running a merge.

**Supported sources:**
- iCloud: Settings → [your name] → iCloud → Contacts → Export vCard
- Proton Mail: Contacts → ⋯ → Export
- Google Contacts: contacts.google.com → Export
- Outlook: File → Open & Export → Import/Export

Files in this folder are read-only inputs — they are never modified.
